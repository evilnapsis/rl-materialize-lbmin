Instrucciones de Instalacion 

1.- Copiar y descomprimir el archivo en la carpeta c:\xampp\htdocs, al final tendras una carpeta llamada “rl-materialize-lbmin”, a la cual accederas desde el navegador como -> http://127.0.0.1/rl-materialize-lbmin
2.- Buscar el archivo schema.sql en rl-materialize-lbmin/schema.sql, una vez localizado copiar y pegar el contenido en una ventana de MySQL, PHPMyAdmin o MySQL WorkBench
3.- En caso de ser necesario modificar el archivo C:\xampp\htdocs\rl-materialize-lbmin\core\controller\Database.php para los datos asociados al acceso a la base de datos.
4.- Acceder a http://127.0.0.1/rl-materialize-lbmin y entrar al sistema con los siguientes datos.
Usuario: admin
Password: admin
